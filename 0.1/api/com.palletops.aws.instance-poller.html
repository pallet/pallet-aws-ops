com.palletops.aws.instance-poller documentation

A poller for instance-ids.

add-instance

fn

[{:keys [config], :as poller} id {:keys [channel notify-when-f remove?], :as options}]

Start polling an id. Writes the instance description to channel when the notify-when-f returns a truthy value when invoked on the instance description. If :remove? is true (the default) the id is removed from the poller when the channel is notified.

add-instances

fn

[{:keys [config], :as poller} idmaps]

Start polling an ids. idmaps is a map from id to a map with :promise and :notify-when-f keys. Notifies promise when the notify-when-f function returns a truthy value when called with the instance.

remove-instance

fn

[{:keys [config], :as poller} id {:keys [channel notify-when-f], :as options}]

Stop polling the specified id. No need to call this if you pass a notify-status to poll.

start

fn

[{:keys [credentials poll-interval api-channel instance-id-map poll-timeout], :or {poller-channel (chan), instance-id-map (atom {}), poll-timeout 10000, poll-interval 2000}, :as options}]

Start a poller using the specified options. :poll-interval is in milliseconds. Returns a map with the poller.

stop

fn

[{:keys [shutdown], :as poller}]

Stop polling
